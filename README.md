# AWS Serverless Hotel API

A CRUD REST API built using AWS Serverless services.

---

## Technologies

- AWS Lambda
- API Gateway
- DynamoDB
- IAM
- Python
- Postman

---

## Features

- Create Hotel
- Get Hotel
- List Hotels
- Update Hotel
- Delete Hotel

---

## API Endpoints

POST /hotels

GET /hotels

GET /hotels/{id}

PUT /hotels/{id}

DELETE /hotels/{id}

---

## Architecture

Client
↓

API Gateway
↓

Lambda

↓

DynamoDB
